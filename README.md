  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jayahendro/sera_test_3.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Running

- Run the project

   ```sh
   npm start
   ```
